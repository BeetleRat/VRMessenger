Подробное описание

Класс создающий аватар пользователя из моделей частей тела.

Данный класс вставляет в SkinnedMeshRenderer шаблона аватара пользователя, модели из полей данного класса.

Аргументы
eyeLeftModelPrefab левого глаза.
eyeRightModelPrefab правого глаза.
headModelPrefab головы.
teethModelPrefab зубов.
outfitTopModelPrefab тела.
hairModelPrefab волос.
