Подробное описание

Класс, хранящий информацию об аватаре.

Данный класс необходим для поиска prefab-ов аватаров в папке Resources. Так же данный класс используется для отображения аватаров в UI.

Аргументы
isActiveЕсли аватар не активен, он не будет добавлятся в список аватаров и не будет отображаться. Это нужно, если какой-то аватар еще не настроен, но необходимо вести работу с другими аватарами.
avatarImageИзображение отображаемое в UI.
avatarNameПуть до аватара в папке Resources/Avatars. По данному пути NetworkManager будет искать данный аватар.
