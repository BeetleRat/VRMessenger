Подробное описание

Класс возвращающий объект в стартовое положение, если объект упал на пол.

Полом является объект имеющий компонент Floor.

Внимание
Объект или его потомки должны иметь коллайдер.
Аргументы
returnSpeedСкорость, с которой объект вернется в исходное положение.
bindObjectОбъекты BackFromFloor, которые должны вернуться на свои места вместе с данным объектом. Это нужно, если у нас сложный объект, в котором несколько независимых/частично зависимых частей. И если одна из таких частей упала на пол - значит необходимо вернуть весь объект в исходное положение. А значит необходимо вернуть в исходное положение все связанные с этим объектом части.
