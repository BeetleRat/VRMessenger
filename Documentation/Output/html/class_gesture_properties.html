Подробное описание

Компонент, хранящий в себе свойства жеста.

Данный компонент, инкапсулирующий работу с информацией о загнутых пальцах в жесте Данный компонент используется для анимации рук на сервере. Каждый жест, используемый для анимации модели руки на сервере, имеет данный компонент.

Аргументы
gestureFingersGestureFingers данного жеста.
