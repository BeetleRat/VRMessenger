Подробное описание

Скрипт обеспечивающий проигрывание анимации моделей рук на сервере

Данный класс обновляет анимацию моделей рук на сервере, в зависимости от жестов распознанных системой GestureAnimation.

Внимание
Для корректной работы данный класс требует, что бы в сцене присутствовали скрипты:
  • ComponentCatcher;
  • NetworkVariables;
  • GestureAnimation;
  • HandsAnimaionUpdater
  • Создано системой 1.9.6
