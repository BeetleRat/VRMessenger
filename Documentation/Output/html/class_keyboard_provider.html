Подробное описание

Класс, обеспечивающий взаимодействие UI с VirtualKeyboardController.

Взаимодействуя с ComponentCatcher, данный класс предоставляет элементам UI взаимодействовать с VirtualKeyboardController.

Внимание
Для корректной работы данный класс требует, что бы в сцене присутствовали скрипты:
  • VirtualKeyboardController;
Аргументы
catcherComponentCatcher, который используется для получения доступа к VirtualKeyboardController.

ComponentCatcher; VirtualKeyboardController; InterfaceHider
