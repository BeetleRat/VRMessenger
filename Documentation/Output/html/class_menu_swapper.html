Подробное описание

Скрипт для переключения различных UI через InterfaceHider

Переключение вкладок меню может быть реализовано различными способами. В данном случае, у нас есть множество UI, каждый из которых это отдельная вкладка меню. Данный скрипт отображает одну из этих вкладок и скрывает все остальные.

Аргументы
menuUIsМассив InterfaceHider, которым управляет данный скрипт.
