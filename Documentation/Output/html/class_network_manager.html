Подробное описание

Класс, отвечающий за взаимодействие с сервером Photon

Данный класс отвечает за:

  • Подключение к серверу;
  • Отключение от сервера;
  • Создание и подключение к лобби;
  • Создание и подключение к комнате;
  • Отключение от комнаты;
  • Отключение от лобби;
  • Отключение от сервера;
  • Спавн игрока на сервере.
Аргументы
vrLoggerVRLoggersManager для вывода логов внутри игры.
sceneChangerSceneChanger для перехода между сценами.
playersPrefabNamestring название prefab-а игрока лежащего в Assets/Resources. Данный prefab будет заспавнен в комнате на сервере при подключении.
defaultRoomsСписок RoomSettings комнат, к которым будет производиться подключение.
autoStartTestRoombool Параметр для отладки. Если true, то автоматически подключает в первую комнату при запуске приложения.
