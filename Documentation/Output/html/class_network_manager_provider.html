Подробное описание

Класс, обеспечивающий взаимодействие UI с NetworkManager.

Взаимодействуя с ComponentCatcher, данный класс предоставляет элементам UI взаимодействовать с NetworkManager.

Внимание
Для корректной работы данный класс требует, что бы в сцене присутствовали скрипты:
  • NetworkManager;
Аргументы
catcherComponentCatcher, который используется для получения доступа к NetworkManager.
См. также
ComponentCatcher; NetworkManager
