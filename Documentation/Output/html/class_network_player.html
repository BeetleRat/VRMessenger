Подробное описание

Класс, отвечающий за синхронизацию локального игрока и его отображения на сервере

Данный класс используется в prefab-е игрока, находящегося по пути Assets/Resources/Avatars.

Данный класс отвечает за:

  • синхронизацию положения головы (шлема oculus);
  • синхронизацию положения рук (контроллеров oculus/отслеживаемых человеческих рук);
  • вид контроллеров (контроллеры oculus/отслеживаемые человеческие руки);
Внимание
Для корректной работы данный класс требует, что бы в сцене присутствовали скрипты:
  • ComponentCatcher;
  • NetworkVariables;
  • HandView;
  • OVRCameraRig;
  • ControllerEvents;
Аргументы
headОбъект отображения головы на сервере (голова в prefab-е игрока).
leftHandОбъект отображения левой руки на сервере (левая рука в prefab-е игрока).
rightHandОбъект отображения правой руки на сервере (правая рука в prefab-е игрока).
controllertTypeControllerМассив ControllerTypeController, которыми управляет данный класс.
dublicateMainPlayerЕсли true, то пользователь будет видеть как его контроллеры отображаются на сервере.
