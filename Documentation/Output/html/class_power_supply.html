Подробное описание

Класс источника электрического тока.

Источник тока постоянно посылает сгенерированный им ток на minus выход.

Аргументы
generatedElectricityElectricity порождаемое данным источником тока.
pluseWireInput источника тока, в который должен входить ток.
minusWireInput источника тока, из которого выходит ток.
См. также
Electricity; WireInput; ElectricalElement; ElectricityTransfer; WireElement;
