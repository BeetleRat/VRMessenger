Класс резистора

По данному классу элемент в электронной цепи определяется как Резистор.

Резистор - пассивный элемент электрических цепей, обладающий определённым или переменным значением электрического сопротивления.

Аргументы
leftInputУсловно левый вход WireInput резистора.
rightInputУсловно правый вход WireInput резистора.
resistanceСопротивление резистора
См. также
WireInput; WireElement; ElectricalElement; ElectricityTransfer
