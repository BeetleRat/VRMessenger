Подробное описание

Компонент, обозначающий данный объект местом для крепления провода

Концы провода содержат компонент WireOutput. Именно он и прикрепляется к данному объекту.

Объекты, содержащие данный компонент, используются для проведения тока, идущего по проводу WireElement к ElectricalElement и наоборот.

Аргументы
plugPositionПозиция, в которую будет прикреплен провод.
unplugPositionПозиция, в которой окажется провод, при откреплении. Данную позицию нужно выбрать так, чтобы при попадании в нее коллайдер провода не соприкасался с коллайдером данного объекта.
breakingDistanceРасстояние, на которое должен отклониться провод от plugPosition, чтобы считать, что он откреплен.
connectedAudioОпциональный параметр. AudioSource воспроизводимый при прикреплении провода.
disconnectedAudioОпциональный параметр. AudioSource воспроизводимый при откреплении провода.
